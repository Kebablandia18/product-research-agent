# argus-sidecar

This folder stores persistent memory for the **Argus** agent.

## Purpose
Competitive intelligence data persistence — scraping patterns, previously analyzed tools, prompt templates, and workflow definitions.

## Files
- `memories.md` — Previously analyzed tools, scraping patterns, session history
- `instructions.md` — Pipeline protocols, analysis frameworks, boundaries

## Folders
- `workflows/` — Complex multi-step workflow definitions
- `knowledge/` — Domain reference material (AI tool landscape)

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/argus/argus-sidecar/`
